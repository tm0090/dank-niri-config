# 🌑 dank-niri-config



## ⚡ Quick Install

> **Requires:** `curl` or `wget` — no `git` needed.

```bash
curl -fsSL https://raw.githubusercontent.com/tm0090/dank-niri-config/main/install.sh | bash
```

The installer is **safe by default** — it will **never silently overwrite** your existing config files.
 

 

## 🛠️ Manual Install

 
```bash
# Clone the repo
git clone https://github.com/tm0090/dank-niri-config.git
cd dank-niri-config

# Copy only — won't overwrite existing files
cp -rn niri ~/.config/
cp -rn DankMaterialShell ~/.config/
```

---

## 📋 Dependencies

Make sure these are installed before applying the config:

- [Niri](https://github.com/YaLTeR/niri) — scrollable-tiling Wayland compositor
- [DankMaterialShell](https://github.com/AvengeMedia/DankMaterialShell) — Desktop shell for wayland compositors
